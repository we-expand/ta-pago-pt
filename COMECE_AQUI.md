## 🎯 PRÓXIMOS PASSOS (3 MINUTOS)

### PASSO 1: Ativar a API (30 segundos)

1. **Abra este link numa nova aba:**
   ```
   https://console.cloud.google.com/apis/library/texttospeech.googleapis.com
   ```

2. Clique no botão azul **"ATIVAR"** (ou **"ENABLE"**)

3. Aguarde 30-60 segundos

4. ✅ Pronto quando vir "API ativada"

---

### PASSO 2: Criar a Chave de API (1 minuto)

1. **Abra este link numa nova aba:**
   ```
   https://console.cloud.google.com/apis/credentials
   ```

2. Clique em **"+ CRIAR CREDENCIAIS"**

3. Selecione **"Chave de API"**

4. **⚠️ COPIE A CHAVE IMEDIATAMENTE!**
   - Formato: `AIzaSyXXXXXXXXXXXXXXXXX...`
   - Guarde num bloco de notas

---

### PASSO 3: Configurar no Sistema (1 minuto)

**OPÇÃO A - Assistente Visual (Recomendado):**

1. Na sua aplicação "Tá Pago.pt", aceda a:
   ```
   /setup/google-tts
   ```

2. Cole a chave no **Passo 4**

3. Clique em **"Validar e Configurar"**

4. Teste a voz PT-PT no **Passo 5**

---

**OPÇÃO B - Via Dashboard:**

1. Entre no sistema com login

2. Vá para: **Campanhas Multicanal** → Aba **"Voz IA"**

3. Clique em **"Configurar Google Cloud TTS"**

4. Cole a chave e valide

---

## 🆘 Problemas?

### Erro: "API Key inválida"
- Aguarde 2-3 minutos após criar a chave
- Verifique se copiou a chave completa

### Erro: "API not enabled"
- Certifique-se de que ativou a API no Passo 1
- Aguarde 1-2 minutos para propagação

### Sem áudio
- Teste noutro navegador
- Verifique ligação à internet
